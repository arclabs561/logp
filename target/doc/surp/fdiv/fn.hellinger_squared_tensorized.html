pub fn hellinger_squared_tensorized(h2: f64, n: usize) -> f64
Expand description

Tensorization of squared Hellinger distance for i.i.d. product measures.

If (H^2(P,Q)) is the squared Hellinger distance between single-sample distributions, then for (n) i.i.d. samples:

H^2(P^{⊗n}, Q^{⊗n}) = 1 - (1 - H^2(P,Q))^n

This identity is the workhorse behind two-point testing lower bounds: if (H^2(P,Q)) is (O(1/n)), then (P^{⊗n}) and (Q^{⊗n}) remain hard to distinguish.

§Arguments

  • h2 - Squared Hellinger distance in [0, 1]
  • n - Number of i.i.d. samples
